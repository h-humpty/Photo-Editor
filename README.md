# Features
- Add photos(title, description, category)
- View photos
- Edit photo details and update photos
- Delete photos
- User registration
- User authentication (Login/Logout)

Set up for development environment
```
npm install (root directory) and 
=> cd frontend
npm install
npm run dev (both client & server)
```
